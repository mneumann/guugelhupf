5. Installation

5.1. Requirements

  • Unix-Umgebung (FreeBSD, Linux oder Windows/Cygwin)

  • MLton

  • Ruby

5.2. Compilieren der gh_* Programme

  1. Compilation Manager installieren:

    % cd src/compilation-manager && su -c cp CM.rb /usr/local/bin

  2. GH_BASE Umgebungsvariable setzen:

    % export GH_BASE=`pwd`

  3. Compilieren unter Unix:

    % cd src/lib/gh/test && make

  4. Compilieren unter Windows/Cygwin:

    % cd src/lib/gh/test && make EXE_EXT=.exe
